PokéDict


PokéDict is a set of dictionaries to be used with the Android app Aard2. They contain translations (actually more like mappings) between Pokémon names, moves, items etc in different languages. The target group is people who grew up playing the Pokémon games in one language but now want to try them in another.



Setup

First, you need to install Aard2 on your device.

Get Aard2

Then, you need to download the dictionary file to your device. Don't worry about ordering, the translations go both ways.

Get .slob file

From the Aard2 app, go to the dictionary tab, press the "+" icon, and locate the .slob file. You're good to go! Catch em all!

If you find any issues while using the dictionary, or if you're missing some words, please open an issue.
